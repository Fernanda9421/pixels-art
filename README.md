## Instruções para clonar o projeto:

1. Clone o repositório

  * `git clone git@github.com:Fernanda9421/pixels-art.git`
  * Entre na pasta do repositório que você acabou de clonar:
    * `cd pixels-art`

2. Você encontrará os arquivos:
   - `index.html`
   - `style.css`
   - `script.js`
